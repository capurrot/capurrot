**🔧 Stack**

```
Frontend  → React + TypeScript
Backend   → Node.js + Express
Database  → PostgreSQL
Deploy    → Docker + Nginx
```

**✨ Highlights**

- 🎵 Algoritmo di raccomandazione musicale
- 🫁 Breathing exercises interattivi
- 📊 Mood tracking nel tempo
- 🔐 Privacy-first architecture
